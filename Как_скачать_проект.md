# Как скачать проект

Для того, чтобы скачать проект к себе на компьютер, нужно нажать на кнопку "Код" ("Clone or download")

![4](./img/4.png)

Или, для дальнейшей работы через терминал (командную строку), можно здесь же скопировать ссылку на проект
